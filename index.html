User Interfaces / User Experience Project 2

Responsive Redesign

About the project

This project required me to choose a website to resdesign from the ground up with html and css. The website could be any one we wished, but it was better to avoid webpages with dedicated UI/UX teams like Instagram or Pintrest. I also had several criterion to consider when deciding how our redesign would improve the webpage, which were broadly grouped together as usability and Accessibility.


My design had to be more resposive than the original page, and had to function on desktop, tablet, and mobile screen sizes.


I then created three low fidelity wireframes in Balsamiq, and a visual style guide and three high fidelity prototypes in Figma.


After that, I was ready to create our redesign in html and css!


The website that I chose, crunchyroll.com, was a promising candidate for redesign.



The original crunchyroll website.


Usability and Accessibility

  • Usability
    • On desktop, the interface is fairly simple to navigate, but the center of the page is packed with content which makes the overall experience cluttered.
    • On mobile or tablet, you’re almost unable to use the website as it doesn’t at all scale for smaller screens.
  • Learnability
    • The interface is simple to learn, as the buttons are mostly simple to discern. However, there are some issues with contrast that makes it difficult to see some elements of the page like the Login button.
  • Memorability
    • The webpage isn’t at all memorable. It feels like a standard, albeit clunky interface, that isn’t inviting for users.
  • Accessibility
    • I used WebAIM WAVE to detect possible accessibility problems.
    • The crunchyroll interface has two main issues: contrast and lack of aria tags.
    • For anyone using a screen reader, the page is essentially impossible to navigate because there are no aria attributes for the screen reader to follow.
    • For people not using a screen reader, there are still many elements on the page that are difficult to see due to a lack of contrasting color. This needs to be fixed.
